# Questo help 
Keywords  [IDE][FIDE][HELP][MAN]

   
 Questo help rappresenta uno strumento che consente di accedere velocemente alla sintassi utilizzata per richiamare una routine senza doverla scrivere   
 o aprire il modulo software per determinare gli IN/OUT  
 Non contiene remark relative al funzionamento 'interno' del modulo, che sono nel modulo stesso, ma solo le informazioni utili per poter richiamare le  
 routine.  
   
 Lo status iniziale indica  
 (0) Informazioni relative a caratteristiche generali di progetto  
 (1) Informazioni relative a librerie  
 (2) Moduli specifici progetto  
 (n) Status successivi secondo utilita' di rappresentazione
