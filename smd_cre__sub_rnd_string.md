# smd_cre::SUB_RND_STRING 
Keywords  [CODESTD]

   
 Restituisce una stringa alfanumerica determinata in modo random utilizzando caratteri maiuscolo, minuscoli e numeri  
 La stringa puo' avere lunghezza definita. Il default e' 10  
   
D_in$ >1 x0                                           
D_in$ >2 Len str desiderata (se 0 --> default 10)     
D_OUT$ > Stringa richiesta                            
> -1 in caso di errore
